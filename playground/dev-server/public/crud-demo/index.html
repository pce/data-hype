Hype — CRUD demos (markup-first)

A small set of markup-first CRUD demos. Each page contains declarative HTML (data attributes + templates). The Hype runtime and optional plugins wire behavior — demos intentionally avoid inline scripts/styles for CSP compatibility.

Available demos

  • Table — classic rows & actions
  • List — stacked list rendering
  • Swipe — horizontal card carousel

Developer notes

These demos expect an ESM Hype runtime at /static/js/hype.js. The external bootstrap module /static/js/crud-demo-bootstrap.js initializes Hype and autowires optional plugins (crud, datatable). This keeps the demo pages minimal and CSP-friendly (no inline scripts or eval).

To run locally: build an ESM Hype runtime, serve it at /static/js/hype.js, then open one of the demo pages above. Inspect the browser console for initialization logs from the bootstrap module.

Markup-first — declare intent in HTML and trust the runtime to wire behavior. Keep demos DRY & KISS.
